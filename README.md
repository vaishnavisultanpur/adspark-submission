# AdSpark

AI-powered ad copy generator for small marketing teams. Drop in a product
brief, pick a brand tone, and get 3 ready-to-use ad variants (headline, body,
CTA) generated by Google Gemini in seconds.

Built for the ThirdEye HR AI Full Stack Web Developer assessment.
See `docs/MASTER_PROMPT.md` and `docs/ARCHITECTURE.md` for the full
requirement analysis, prompt engineering, and architecture writeup.

## Stack
Next.js 14 (App Router, TypeScript) · Tailwind CSS · NextAuth (Credentials +
JWT) · Neon Postgres (Prisma ORM) · Zod · Google Gemini API

## Run locally

```bash
npm install
cp .env.example .env.local   # fill in DATABASE_URL, DIRECT_URL, GEMINI_API_KEY, NEXTAUTH_SECRET
npx prisma generate
npx prisma migrate dev --name init
npm run dev
```

Open http://localhost:3000, register an account, create a campaign, and click
**Generate ad copy**.

## Build

```bash
npm run build
npm run start
```

## Environment variables

| Variable | Description |
|---|---|
| `DATABASE_URL` | Neon pooled Postgres connection string |
| `DIRECT_URL` | Neon direct Postgres connection string (used for migrations) |
| `GEMINI_API_KEY` | Gemini API key from aistudio.google.com/app/apikey |
| `NEXTAUTH_SECRET` | Random secret for signing session JWTs — `openssl rand -base64 32` |
| `NEXTAUTH_URL` | App URL (`http://localhost:3000` locally; your deployed domain in production) |

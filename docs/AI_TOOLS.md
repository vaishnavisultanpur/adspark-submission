# AI Tools Used

## Claude

Claude was used for:

- Requirement analysis
- Product planning
- Prompt engineering
- UI/UX suggestions
- Code generation
- Debugging
- Refactoring
- Documentation preparation

## Google Gemini

Google Gemini API was integrated into the application to generate advertisement content.

The model generates:

- Headline
- Advertisement Body
- Call-To-Action (CTA)

Three advertisement variants are generated for every campaign.
